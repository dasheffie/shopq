# ShopQ Progress Log

## Codebase Patterns
- Server: Minimal Express serving static files (use server-utils.js)
- Frontend: Single index.html with inline <style> and <script>
- Storage: All data in localStorage, JSON stringify/parse
- Categories: Predefined array with id, name, emoji, color
- Share: Base64 encode list data in URL query param
